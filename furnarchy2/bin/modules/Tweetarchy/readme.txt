Tweetarchy v1.1
A Furnarchy module by Mercenary Enclave Productions

Tweetarchy is a module that acts as a basic Twitter client. It can display incoming tweets in the Furcadia client window, as well as send them with a simple command.

Commands:
@tweet [message] - Tweet the following text. ("@tweet Hi, I'm on Furcadia!")
