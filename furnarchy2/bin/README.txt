II. ABOUT FURNARCHY

Furnarchy is a free, open source, pluggable proxy program
for the Furcadia client.

For more information, to report bugs, make suggestions, 
etc., visit the Furnarchy forums at:
      
         http://proxy.entwinedstudios.com/forums
         
Or contact the authors at:

         webmaster@entwinedstudios.com
         webmaster@mercenary-enclave.com
